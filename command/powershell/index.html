      PowerShell

      Το PowerShell είναι εργαλείο της Microsoft που βρίσκεται ήδη προεγκατεστημένο στα Windows 10 και δίνει τη δυνατότητα στους διαχειριστές συστημάτων να ελέγξουν κάθε πτυχή του λειτουργικού μέσω εντολών και scripts.

      Εκτελεί μια επισκευή συστήματος για να εντοπίσει και να επιδιορθώσει τυχόν αρχεία συστήματος που έχουν καταστραφεί.

      Εντολή Περιγραφή
      Get-Service Εμφάνιση πληροφοριών σχετικά με τα Services
      Get-Process Εμφάνιση πληροφοριών σχετικά με τα processes
      Get-EventLog Εμφάνιση πληροφοριών από αρχεία καταγραφής συμβάντων
      Set-ExecutionPolicy Τροποποιήστε την πολιτική εκτέλεσης σεναρίου
      Test-Connection Ελέγξτε τη συνδεσιμότητα με έναν απομακρυσμένο υπολογιστή
      Out-File Αποθήκευσε την έξοδο του PowerShell σε ένα αρχείο
      Get-Help Εμφάνιση πληροφοριών σχετικά με έννοιες και εντολές
      Get-History Ανακτήστε πρόσφατες εντολές.
      Get-Command Ανακτήστε όλες τις διαθέσιμες εντολές PowerShell
      ConvertTo-HTML Δημιουργήστε ένα αρχείο HTML από την έξοδο PowerShell
      Copy-Item Αντιγράψτε ένα αρχείο σε μια συγκεκριμένη τοποθεσία
      Clear-History Διαγράψτε τις εγγραφές από το ιστορικό εντολών σας
      Add-History AΠροσθέστε καταχωρήσεις στο ιστορικό εντολών σας
      Format-Table Μορφοποιήστε την έξοδο PowerShell ως πίνακα
      Format-List Μορφοποιήστε την έξοδο PowerShell ως λίστα
      Clear-Content Διατηρήστε ένα στοιχείο ενώ διαγράφετε τα περιεχόμενα αυτού του στοιχείου
      Checkpoint-Computer Ορίστε ένα σημείο επαναφοράς στο μηχάνημά σας
      ForEach-Object Εκτελέστε μια λειτουργία σε κάθε στοιχείο σε ένα καθορισμένο group
      Where-Object Επιλέξτε αντικείμενα με μια συγκεκριμένη ιδιότητα
      Select-Object Επιλέξτε συγκεκριμένες ιδιότητες ενός αντικειμένου ή μιας ομάδας αντικειμένων
      Write-Progress Εμφανίστε μια γραμμή προόδου σε ένα παράθυρο PowerShell
      Debug-Process Επισυνάψτε ένα πρόγραμμα εντοπισμού σφαλμάτων σε μια διαδικασία που εκτελείται
      Get-WinEvent Εμφάνιση αρχείων καταγραφής συμβάντων των Windows
      Wait-Job Καταργήστε τη γραμμή εντολών μέχρι να ολοκληρωθεί η εκτέλεση των εργασιών παρασκηνίου

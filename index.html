      Λειτουργικό Σύστημα Windows

      Τα Windows είναι το λειτουργικό σύστημα της Microsoft είναι ο τρόπος επικοινωνίας με τον υπολογιστή μας . Μας παρέχει ένα γραφικό περιβάλλον (γνωστό και ως Graphical User Interface (GUI)) μέσω του οποίου μπορούμε να διαχειριστούμε και να εκτελέσουμε διάφορες λειτουργίες στον υπολογιστή μας.

      Εργαλεία Διαχείρισης Συστήματος

      Τα Windows παρέχουν διάφορα εργαλεία διαχείρισης συστήματος που επιτρέπουν στους χρήστες και τους διαχειριστές να διαμορφώνουν, να παρακολουθούν και να επιλύουν προβλήματα στον υπολογιστή.
      Μερικά από τα εργαλεία διαχείρισης συστήματος στα Windows είναι:

      1. Πίνακας Ελέγχου (Control Panel)
        Είναι το κεντρικό μέρος όπου οι χρήστες μπορούν να διαμορφώσουν διάφορες ρυθμίσεις του συστήματος, όπως δίκτυο, ήχο, εμφάνιση, κλπ.
      2. Διαχείριση Συστήματος (System Management)
        Περιλαμβάνει πληροφορίες για τον υπολογιστή.
      3. Διαχείριση Συσκευών (Device Manager)
        Μπορούμε διαχειριστούμε τις συσκευές που είναι συνδεδεμένες στον υπολογιστή.
      4. Διαχείριση Διακομιστή (Server Manager)
        Χρησιμοποιείται σε Windows Server για τη διαχείριση των Server και των υπηρεσιών.
      5. Διαχείριση Υπηρεσιών (Services Management)
        Επιτρέπει τη διαχείριση των υπηρεσιών του υπολογιστή.
      6. Διαχείριση Σκληρών Δίσκων (Disk Management)
        Επιτρέπει τη διαμόρφωση και τη διαχείριση συσκευών αποθήκευσης (πχ σκληρών δίσκων).

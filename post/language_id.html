    Language ID

    Οι κωδικοί LCID (ή Language ID) χρησιμοποιούνται κυρίως στα λειτουργικά συστήματα της Microsoft, και συγκεκριμένα στα Windows, για την αναγνώριση και τη διαχείριση διαφορετικών γλωσσών και περιοχικών ρυθμίσεων. Αυτό επιτρέπει στα λειτουργικά συστήματα Windows να προσαρμόζουν την εμφάνιση και την αλληλεπίδραση με τον χρήστη σύμφωνα με τις προτιμήσεις του.
